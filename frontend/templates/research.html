Model Architecture

Our system uses InceptionResNetV2, a state-of-the-art deep learning model pre-trained on ImageNet and fine-tuned for oral cancer detection.

Input Image (299x299)
InceptionResNetV2
Global Pooling

Key Features

  • Transfer Learning: Pre-trained on ImageNet
  • Grad-CAM Visualization: Explainable AI
  • Data Augmentation: Rotation, Flip, Zoom
  • Validation: Stratified K-Fold

Dataset Information

The model is trained on a comprehensive dataset of histopathology images containing both cancerous (Oral Squamous Cell Carcinoma) and non-cancerous tissue samples.
